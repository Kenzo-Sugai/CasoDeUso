Description

Visualizar Agendamento

Atores: Cliente

Sumário: O caso de uso descreve o processo do cliente que irá visualizar o agendamentos que já realizou. O cliente já deve estar com o agendamento feito.

Pré-Condições: O cliente deve já ter um cadastro feito, e também que ele tenha já feito um agendamento.

Pós Condições: O agendamento é envia para o banco de dados, isso permite que o cliente possa ver os agendamentos que ele já fez.

Ponto de Inclusão: Não a nada para incluir.

Ponto de Extensão: Não a nada para incluir.

Fluxo principal: 1. O cliente entra em agendamentos. 2. O cliente entra na opção de visualizar meus agendamentos. 3. Após isso o sistema retorna todos os agendamentos já feito.
