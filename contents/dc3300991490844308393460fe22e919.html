Description

Cancelar Agendamento

Atores: Cliente

Sumário:
Este caso de uso descreve o processo de cancelar um agendamento já realizado anteriormente.

Pré-condições:
O administrador deve estar logado em sua conta no aplicativo e ter permissão para gerenciar os agendamentos das quadras.

Pós-condições:
O agendamento deve ser concluído com sucesso. O proprietário deve ser notificado sobre a alteração do agendamento feito pelo cliente.

Ponto de Inclusão:
Não há nenhum Ponto de Inclusão

Ponto de Extensão:
Não há nenhum Ponto de Extensão

Fluxo Principal: 1. O Cliente deve acessar seu agendamento na página de visualização de agendamentos. 2. O Cliente deve clicar na opção de cancelar agendamento. 3. O Cliente deve confirmar a opção de cancelar agendamento. 4. Aparecerá notificação de cancelamento e o software irá direcioná-lo para a página de visualização de agendamentos.
