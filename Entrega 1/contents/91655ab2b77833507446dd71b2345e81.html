Description

Contratar Planos Mensais

Atores: Cliente

Sumário: Este caso de uso descreve o processo de um cliente realizando uma assinatura de plano mensal através do sistema.

Pré-condições:
Para que o cliente possa assinar, deve ter uma conta válida no aplicativo. Além disso deve estar logado em sua conta no momento da assinatura do plano.

Pós-condições:
O plano deve ser ativado com sucesso para a conta do cliente. Além disso deve receber acesso às funções exclusivas do plano imediatamente.

Ponto de Inclusão:
Não há nenhum Ponto de Inclusão

Ponto de Extensão: Não há nenhum Ponto de Extensão

Fluxo Principal: 1. O cliente acessa a seção de planos no aplicativo. 2. O cliente visualiza informações sobre os benefícios e as opções de planos disponíveis. 3. O cliente seleciona o plano que deseja assinar. 4. O sistema leva o cliente para a página de pagamento do plano. 5. O cliente efetua o pagamento do plano. 6. O sistema confirma o pagamento e ativa o plano premium para a conta do cliente. 7. O sistema exibe as funções exclusivas disponíveis para os clientes que assinaram o plano. 8. O cliente pode desfrutar das funções exclusivas durante todo o período de validade do plano.

Fluxo Alternativo: 4.1 - Se o pagamento da assinatura do plano for recusado pelo sistema de pagamento, o sistema informa o cliente e o leva de volta para a página de pagamento para que ele possa corrigir as informações do pagamento ou selecionar um método de pagamento diferente.

5.1 - Se não houver planos disponíveis para a região do cliente, o sistema informa o cliente e sugere outros planos disponíveis que possam atender às necessidades dele. Caso não haja outros planos disponíveis, o sistema pode oferecer a opção de se cadastrar para receber uma notificação quando o plano estiver disponível na região do cliente.

Fluxo de Exceção: 8.1 - Se ocorrer um problema durante a ativação do plano premium, o sistema informa o cliente sobre a falha. O sistema pode oferecer algumas possíveis soluções para o problema, como tentar novamente em alguns minutos, verificar as informações de pagamento ou entrar em contato com a equipe de suporte para obter ajuda.
